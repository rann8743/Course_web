
# Course Website

This is a web project for a course at the University of Toronto Scarborough. The website provides information about the course syllabus, team, lectures, labs, and assignments.

## Introduction
This project aims to provide a comprehensive overview of the course. The website is built using React.js and styled with CSS. It includes sections for the course syllabus, team, lectures, labs, and assignments.

## Features
- Fixed navigation bar with links to different sections
- Detailed syllabus information
- Course team details
- Weekly lecture topics and materials
- Lab topics and materials
- Assignment PDFs and LaTeX files

## Technologies
- React.js
- CSS
- HTML

## Setup
To set up the project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/course-website.git
    cd course-website
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm start
    ```

The website will be available at `http://localhost:3000`.

## Usage
Navigate through the website using the navigation bar to access different sections. Each section provides detailed information related to the course.
